# ✈ AI Flight Recommendation Agent

An AI-powered Flight Recommendation System that provides live flight recommendations using **Google Flights (SerpAPI)**. The application allows users to search flights based on source, destination, travel dates, budget, and preferred airline.

---

## 🚀 Features

- 🌍 Live Flight Search using Google Flights API (SerpAPI)
- 💰 Budget-Based Flight Recommendation
- ✈ Preferred Airline Selection
- 📅 Departure & Return Date Selection
- ⚡ FastAPI Backend
- 🎨 React Frontend
- 🔄 REST API Integration
- 📱 Responsive UI

---

## 🛠 Tech Stack

### Frontend
- React.js
- Axios
- HTML5
- CSS3

### Backend
- FastAPI
- Python
- Requests
- Python-dotenv

### API
- Google Flights API (SerpAPI)
